## Decisions log (so they are not relitigated)
- **Official reference class = the synthetic field** тАФ a recency-weighted topic-mixture blend of full (subfield, vintage) cohorts (reference-based тЖТ community migration). The cited-only co-citation *neighborhood* is **not** a ranking cohort (it's popularity-biased and has no uncited papers); it only informs the topic mixture. (QaL_spec ┬з3/┬з5.)
- **Denominator rule:** every percentile is over the **full population** (cited + uncited), never cited-only and never a top-N/sorted sample; the uncited atom takes the **mid-rank 100┬╖pтВА/2**; the same convention is used for r_obs and rтИЮ.
- Calibration seed = Decision Sciences subfields **1803, 1802, 1800** (extensions 1405, 1804 available).
- QaL = calibrated posterior over eventual percentile (point + 90% interval + bucket probs); back-tested to ~90% coverage via per-age split-conformal.
- **No single author-level score** (author page shows a per-paper distribution).
- Stack = GitHub (batch) + Vercel/Next.js (web + read API) + Neon Postgres; **Vercel does no heavy compute**.
